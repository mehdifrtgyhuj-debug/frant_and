# frant_and
این پروژه یک وب‌سایت فروشگاهی مدرن و واکنش‌گرا برای ارائه و فروش لوازم دیجیتال است که با استفاده از تکنولوژی‌های HTML5، CSS3 و JavaScript (Vanilla) توسعه داده شده است.  هدف اصلی این پروژه، ایجاد یک تجربه کاربری سریع، روان و حرفه‌ای برای کاربران در هنگام مرور و انتخاب محصولات دیجیتال مانند موبایل، لپ‌تاپ، هدفون و سایر تجهیزات الکترونیکی می‌باشد.
